# High Style

Re-usable style components free you up to create high-quality component libraries and design systems, and retil-css facilitates this too with **high style** â€“ style objects that allow for different values in different surfaces.

```jsx
import { highStyle } from 'retil-css'
import { createSurfaceSelector, inHoveredSurface } from 'retil-interaction'
import { media } from 'retil-media'

// High Style works with custom surface selectors too!
const inCheckedSurface = createSurfaceSelector('[aria-checked="true"]')

<div css={highStyle({
  color: {
    default: 'black',
    [inHoveredSurface]: 'red',
    [media.small]: {
      [inCheckedSurface]: 'black',
    }
  },
  fontSize: {
    default: '16px',
    [media.small]: '14px',
  }
})} />
```
